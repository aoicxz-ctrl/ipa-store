📌 الطريقة الأولى
  1. تأكد أنك تمتلك شهادة مدفوعة.
  2. افتح الملف وقم بمشاركته لأي برنامج توقيع تطبيقات IPA مثل Ksign أو Gbox أو Esign أو غيرها.
  3. سيفتح الملف في التطبيق، اضغط عليه واختر (توقيع وتثبيت) أو Sign & Install.
🤖 الطريقة الثانية (بوت التليجرام)
  1. اذهب لهذا البوت @ipasignxbot.
  2. اضغط Add IPA.
  3. اضغط Add Cert وأضف ملفات شهادتك والباسورد.
  4. أرسل للبوت ملف الـ IPA واضغط Sign.
  5. بعدها اضغط Install ثم Install Now.
